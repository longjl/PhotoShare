# PhotoShare

#Gradle 版本 : 2.3

#构建

#第一步:
        $ git clone https://github.com/longjl/PhotoShare.git PhotoShare

        $ cd PhotoShare

        $ git init

#第二步:
        $ cd PhotoShare/deps/

        $ git submodule init

        $ git submodule update
